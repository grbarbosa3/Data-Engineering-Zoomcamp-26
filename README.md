# Data Engineering Zoomcamp 2026

This repository documents my journey through the **DataTalksClub Data Engineering Zoomcamp 2026**, a free hands-on course focused on the fundamentals of data engineering and modern data pipelines.

Throughout the course, I am building practical experience with tools and concepts such as **Docker, PostgreSQL, Terraform, orchestration, data warehousing, dbt, Spark, and Kafka**, while also documenting my progress module by module.

The goal of this repository is to serve as both:
- a personal knowledge base
- a public record of my learning journey in Data Engineering

## About the course

The **Data Engineering Zoomcamp** is a practical course that covers the end-to-end foundations of data engineering, from local environments and infrastructure setup to batch and streaming pipelines.

It includes topics such as:
- containerization
- infrastructure as code
- workflow orchestration
- data warehousing
- analytics engineering
- batch processing
- streaming systems
